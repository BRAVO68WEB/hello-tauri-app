A simple Rust app that uses the [Tauri](https://tauri.app) Framework from [Rust](https://www.rust-lang.org/).

It is an open source application that can be used with desktop os.

## Dev

Run the app with 
```bash
yarn tauri dev
```

## Build 

Build the app with 
```bash
yarn tauri build
```
